Для запуска блока CIC в Vivado HLS необходимо выполнить следующие шаги:

1. Запустить Vivado HLS 2016.4 Command Promt;
2. В командной строке выбрать директорию, в которой находится
   скрипт run.tcl : cd "dir"
   (В данном случае cd "D:\Repositories\hardware\src\Vivado-HLS\CIC");
3. Запустить скрипт: "vivado_hls run.tcl"
